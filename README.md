# Moviemania
Description:-
Moviemania a IMDB clone website. <br>

Live Demo of Moviemania:- https://skmoviemania.netlify.app
<br>

Technologies used:- HTML, CSS, SASS and JavaScript. 
<br>

<h3>Project Screenshorts:-</h3> 

<b>Home Page</b><br>

Functionality :-<br>

<ul>
  <li>Search any movies from search bar.</li>
  <li>Add movie to your favorite.</li>
  <li>Navigate to favorite page from heart icon on navbar.</li>
</ul> 

![Home Page](https://user-images.githubusercontent.com/72254047/217007445-950861ca-4ab0-4460-b140-80482286228e.png)

<b>Favorite Page</b><br>

Functionality :-<br>

<ul>
  <li>Navigate back to home page from back icon on navbar.</li>
  <li>Delete movie from favorite separately by delete icon.</li>
  <li>Delete all movies favorite page by clear all button.</li>
</ul>
